# HoopStats

HoopStats is a basketball-focused web application that displays players, schedules, and team-related content.

## Tech Stack
- HTML
- CSS
- JavaScript

## Features
- Multi-page navigation (Players, Schedules, News, Chat)
- Responsive layout
- Image-based content rendering
